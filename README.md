# Why Cloudflare Workers?

Instead of deploying a traditional Express backend, the AI layer executes entirely through Cloudflare Workers.

This architecture enables:

- edge execution
- lower latency
- secure API key management
- frontend/backend separation
- GitHub Pages compatibility
- zero backend hosting overhead

The result is a scalable AI-powered portfolio without requiring a traditional Node deployment.
